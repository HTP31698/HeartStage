# HeartStage 개발 기록

## UI 시스템

### ToastUI (2025-12-20)
범용 토스트 알림 시스템

**파일:**
- `Assets/Scripts/UI/Common/ToastUI.cs` - 메인 싱글톤
- `Assets/Scripts/UI/Common/ToastItemView.cs` - 개별 토스트 뷰
- `Assets/Prefabs/UI/ToastCanvas.prefab` - 캔버스 프리팹
- `Assets/Prefabs/UI/ToastItem.prefab` - 토스트 아이템 프리팹

**특징:**
- DontDestroyOnLoad (씬 전환 시 유지)
- 동시 표시 1개 (새 메시지 오면 기존 것은 위로 슬라이드 후 교체)
- 슬라이드 다운 + 페이드 애니메이션 (DOTween)
- 다크 그레이 배경 `rgba(0.12, 0.12, 0.12, 0.8)`
- RaycastTarget = false (터치 안 막음)
- 2줄 텍스트 지원
- 상단에서 350px 아래 위치

**사용법:**
```csharp
ToastUI.Show("메시지");
ToastUI.Show("메시지", 3f);  // 3초 표시
ToastUI.Success("성공 메시지");
ToastUI.Warning("경고 메시지");
ToastUI.Error("에러 메시지");
ToastUI.Info("정보 메시지");
```

**적용된 곳:**
- PurchaseConfirmPanel - 구매 완료 토스트
- FriendAddItemUI - 친구 신청 결과
- FriendRequestItemUI - 친구 수락/거절 결과
- FriendSentRequestItemUI - 요청 취소 결과

### NoteLoadingUI (2025-12-20)
노트/악보 로딩 인디케이터

**파일:**
- `Assets/Scripts/UI/Common/NoteLoadingUI.cs`
- `Assets/Prefabs/UI/NoteLoadingCanvas.prefab`

**특징:**
- DontDestroyOnLoad (씬 전환 시 유지)
- 싱글톤 패턴
- BeginLoading() / EndLoading() 카운터 방식

**사용법:**
```csharp
NoteLoadingUI.Instance.BeginLoading();
// ... 로딩 작업 ...
NoteLoadingUI.Instance.EndLoading();
```
