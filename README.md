## 🎮 게임 소개

`하트스테이지`는
**악마에 홀린 사람들을 아이돌 공연으로 정화한다**는 컨셉의
2D 캐주얼 디펜스 게임입니다.

플레이어는 스테이지 입장 전 캐릭터를 배치하여
캐릭터 간 **시너지 및 패시브 효과**를 전략적으로 구성하고,
웨이브 동안 몰려오는 적들을 막아내며 스테이지를 클리어합니다.
